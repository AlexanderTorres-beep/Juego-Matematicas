## 📋 Descripción

**Aventura de los Números Mágicos** es una aplicación web educativa diseñada para que los niños de 6-7 años aprendan matemáticas de forma divertida e interactiva. No requiere instalación, funciona en cualquier navegador y está optimizada para tablets, celulares y computadoras.

### ✨ Características Principales

- 🍎 **4 mini-juegos educativos**: Conteo, Sumas, Restas y Secuencias
- 🎯 **Dificultad progresiva**: Se adapta automáticamente al nivel del niño
- ⭐ **Sistema de recompensas**: Estrellas y celebraciones para motivar
- 📱 **Diseño responsive**: Funciona perfecto en celulares y tablets
- 🎨 **Interfaz colorida**: Atractiva visualmente para niños pequeños
- 🔊 **Retroalimentación inmediata**: Respuesta visual al acertar o intentar
